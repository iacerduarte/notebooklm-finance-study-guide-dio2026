## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da **DIO (Digital Innovation One)**, durante a aula do **Tech Educator Filipe Aguiar**, com o objetivo de explorar o potencial do **Google NotebookLM** na construção de um **Segundo Cérebro** (Second Brain) baseado em fontes confiáveis sobre investimentos.

A proposta consiste em reunir conteúdos de diferentes fontes para que o NotebookLM seja capaz de responder perguntas utilizando apenas as informações fornecidas em sua base de conhecimento, reduzindo significativamente o risco de **alucinações** (respostas inventadas por modelos de IA).

O tema escolhido foi a filosofia de investimentos de **Benjamin Graham**, considerado o pai do *Value Investing*.

---

# 🎯 Objetivo

Criar um **Segundo Cérebro** especializado na metodologia de investimentos de **Benjamin Graham**, utilizando o Google NotebookLM como ferramenta de organização e consulta de conhecimento.

Ao alimentar o NotebookLM apenas com fontes confiáveis, a IA passa a responder perguntas fundamentadas nas referências fornecidas, tornando as respostas mais consistentes e próximas do conteúdo original estudado.

# 🏗️ Construção do NotebookLM

Após reunir as fontes, foi criado um novo caderno no NotebookLM.

### Prompt Inicial

```
Comporte-se como um segundo cérebro baseado nos ensinamentos do Benjamin Graham e sua metodologia de investimentos.
```

Esse prompt foi inserido em:

```
+ Adicionar Fonte
→ Texto Copiado
→ Colar Texto
→ Inserir
```

---

# 🔍 Pesquisa com Deep Research

Além das fontes previamente selecionadas, foi utilizada a funcionalidade **Pesquisa Rápida + Deep Research** do NotebookLM.

Consulta utilizada:

```
metodologias de investimentos de Benjamin Graham
```

O objetivo foi complementar automaticamente a base de conhecimento com novas referências relacionadas ao tema.

# 📌 Miniguia de Utilização

## 1️⃣ Criar um novo Notebook

Acesse o Google NotebookLM e crie um novo projeto.

---

## 2️⃣ Adicionar as fontes

Inclua:

- vídeos
- textos
- PDFs
- links

---

## 3️⃣ Definir um contexto

Adicione um texto orientando o comportamento da IA.

Exemplo:

```
Comporte-se como um segundo cérebro baseado nos ensinamentos do Benjamin Graham e sua metodologia de investimentos.
```

---

## 4️⃣ Complementar a base

Utilize o recurso **Deep Research** para encontrar novas referências sobre o tema.

---

## 5️⃣ Validar o conhecimento

Faça perguntas sobre o conteúdo para verificar se a IA responde utilizando apenas as fontes cadastradas.

# ✅ Resultados Obtidos

Durante os testes foi possível observar que:

- as respostas permaneceram consistentes com as fontes adicionadas;
- o NotebookLM reduziu significativamente respostas inventadas (alucinações);
- a organização do conhecimento facilitou consultas rápidas;
- o recurso Deep Research complementou a base de conhecimento com novas informações relevantes.
